ATOM
===
```
¿Porque atomon?
- ATOM presenta infinita libertad debido a que es un editor libre de codigo abierto.
- Existen muchos paquetes disponibles que hacen que no se tenga nada que envidiar a editores como PHPStorm, Sublime, Eclipse.
```
INSTALACIÓN
==
```
$ curl -L https://packagecloud.io/AtomEditor/atom/gpgkey | sudo apt-key add -
$ sudo sh -c 'echo "deb [arch=amd64] https://packagecloud.io/AtomEditor/atom/any/ any main" > /etc/apt/sources.list.d/atom.list'
$ sudo apt-get update
$ sudo apt-get install atom
```

PAQUETES RECOMENDADOS
==
```
Autocompletar código php que tiene dependencias composer
atom-autocomplete-php

Ir a la definición de una variable, clase o función
goto-definition

Hacer ctrl+click en un texto y te envia a su definición (ctrl+shift desde teclado)
hyperclick
js-hyperclick
php-hyperclick
```
