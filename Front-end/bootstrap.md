Bootstrap
========
#### Sistema de grilla
Tamaños disponibles
```
Extra small <576px	
Small ≥576px
Medium ≥768px
Large ≥992px
Extra large ≥1200px
```
Clase para filas
```
row
```
Clases para columnas

| Extra small | Small    | Medium   | Large    | Extra large |
| ----------- | -------- | -------- | -------- | ----------- |
| .col-       | .col-sm- | .col-md- | .col-lg- | .col-xl-    |

#### Responsivo
Tamaños responsivos de Bootstrap 4
```
Display 1 (6rem = 90px)
Display 2 (5.5rem = 82.5px)
Display 3 (4.5rem = 67.5px)
Display 4 (3.5rem = 52.5px)
h1 (2.5rem = 40px)
h2 (2rem = 32px)
h3 (1.75rem = 28px)
h4 (1.5rem = 24px)
h5 (1.25rem = 20px)
h6 (1rem = 16px)
p (1rem = 16px)
```
Volver responsivo el h1
```
h1 {font-size:1rem;} /*1rem = 16px*/
```
