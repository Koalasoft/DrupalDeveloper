VISUAL STUDIO CODE
===
```
¿Porque Visual Studio Code?
Es ligero y presenta todas las funcionalidades que a un programador le ahorran tiempo y esfuerzo:
- Autocompleta código: php, js, html, css
- Reconoce el formato del código: php, js, html, css, twig, htaccess
- Soporta snipets.
- Marca el versionamiento del código git.
- Con ctrl-click se dirige directamente al código que implementa una funcionalidad.
- Cuando se escribe código autocompleta las librerías que lo contienen PHP-Drupal
```
